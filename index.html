INTRODUÇÃO A INFORMÁTICA

Informática: É a ciência que estuda o processamento automático da informação por meio do computador.

Computador: É uma máquina electrónica que recebe dados através dos periféricos de entrada, processa esses dados realizando operações lógicas e aritméticas para resolver um determinado problema, transformando-os em outros dados a que se chama de informação. Tal informação é enviada aos periféricos de saída ou simplesmente armazenada. O computador permite realizar tarefas ou resolver problemas em menos tempo e com menos esforço. Permite também reduzir espaço para armazenamento de Informação.

INTRODUÇÃO A LINGUAGEM DE PROGRAMAÇÃO

As linguagens humanas (como por exemplo o Português, Francês, Inglês, etc), servem para a comunicação entre dois indivíduos. No caso das linguagens de programação servem para estabelecer a comunicação entre o Homem e o computador, permitindo assim o processamento automático da informação.

Linguagem: É um código de comunicação formado por léxico, sintaxe e semântica.

Léxico>: É um conjunto de vocábulos disponíveis numa linguagem.

Sintaxe>: É a forma de escrever os vocábulos.

Semântica: É o significado da combinação dos vocábulos.

Linguagem de Programação: É um conjunto de palavras reservadas e regras, sintácticas e semânticas, usadas para descrever a execução de um algoritmo no computador.

Programar: É criar um conjunto de instruções ou comandos numa sequência lógica, que o computador interpreta para poder executar uma tarefa ou resolver um determinado problema. Essas instruções são escritas utilizando uma determinada linguagem de programação.

TIC (Tecnologias da Informação e Comunicação)

A necessidade do homem se comunicar já existe a muitos séculos. Começou com sinais de fumo usado pelos índios, aparecendo depois o pombo-correio (Nos tempos em que os reis travavam grandes batalhas para conquistarem grandes territórios, além dos mensageiros usavam também o pombo-correio para levar mensagens. Aproveitando-se da característica que os pombos sempre voltam ao seu próprio ninho), telefone de dois copos e um fio ligados entre eles, tambores, Cornetas de chifres, cartas, etc. Com o aparecimento das tecnologias a comunicação da informação tornou-se mais fácil e mais rápida.

TIC: É o conjunto das tecnologias que permitem a comunicação da informação entre as pessoas por meio de equipamentos tecnológicos, como por exemplo: Celular, Rádio, Televisão, Computador, etc.
